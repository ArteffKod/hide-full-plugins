# How to hide full your minecraft plugins

SOON BY ARTEFFKODS

There are 4 ways to view plugins

- Commands method: `/pl` `/bukkit:pl` etc.
- Tab method: `/ver [TAB]` `/about <TAB>`
- TAB method: [TAB]:{commands}
- Query plugins method: `bukkit.yml`
